# MultiVibing

MultiVibing is a web interface for Codex CLI. The app is served by a Go HTTP
server with a Vite/React frontend and a Codex app-server gateway.

## Development

```bash
npm install
npm run dev
```

`npm run dev` starts the Vite frontend and the Go web server, then opens the
local browser.

## Verification

```bash
npm test
```
