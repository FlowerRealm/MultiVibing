# MultiVibing

MultiVibing is a local web app for opening projects and running project-scoped
terminals. The app is served by a Go HTTP server with a Vite/React frontend.

## Development

```bash
npm install
npm run dev
```

`npm run dev` starts the Vite frontend and the Go web server. Open the local URL
shown in the terminal to view the app.

## Verification

```bash
npm test
```
